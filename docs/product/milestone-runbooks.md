# Milestone runbooks

Milestone runbooks are feature-by-feature validation guides for release and
pilot testing. They are operational artifacts, but they belong with product
planning because they explain what each milestone is expected to prove.

| Milestone | Runbook |
| --- | --- |
| 0.4.0 alpha | [Milestone 0.4 test runbook](../operations/milestone-0-4-test-runbook.md) |
| 0.5.0 alpha | [Milestone 0.5 test runbook](../operations/milestone-0-5-test-runbook.md) |

Use these runbooks with sandbox projects and redacted configuration snippets.
Do not attach provider tokens, license keys, private keys, customer data, or raw
state files to issue reports.
